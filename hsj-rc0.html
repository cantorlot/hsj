Instructions

Uses number keys 1-4 to play.
Use 9 and 0 for player 2.
Use up arrow and right arrow for player 3.

Missing in-game instructions

  • Super mode needs to be toggled in the options menu after it is activated
  • Abilities in super mode need to be triggered. Pinkie Pie has the easiest trigger.
  • Twilight Sparkle can teleport back to the start by holding 2. She can also bank moves (press the keys for many moves and she will execute them in order).
